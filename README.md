# Mohamed Samir Ali El Feky - Portfolio

A premium, executive-level personal portfolio website showcasing skills in media buying, marketing strategy, and CMO experience.

## Deployment to GitHub Pages

### Step 1: Create GitHub Repository
1. Go to GitHub and create a new repository named `mohamed-samir-portfolio`.
2. Make it public.
3. Do not initialize with README, .gitignore, or license.

### Step 2: Upload Files
Upload the following files to the repository:
- `index.html`
- `style.css`
- `script.js`
- `assets/` (if any)

### Step 3: Enable GitHub Pages
1. Go to the repository Settings.
2. Scroll to Pages section.
3. Under Source, select "Deploy from a branch".
4. Branch: `main`, Folder: `/ (root)`.
5. Save.

### Step 4: Access Your Site
After a few minutes, your site will be live at:
`https://[your-username].github.io/mohamed-samir-portfolio/`

## Local Development
Open `index.html` in a web browser to view locally.
